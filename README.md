# 🦊 SuzuranBot - “我们的光”

这是一个基于《明日方舟》干员“铃兰”（Suzuran）的 LLM (大语言模型) 角色扮演（Roleplay）上下文与设定仓库。本项目收集了铃兰的详尽背景设定、干员档案和语音台词，并以此提炼出了一套高质量的 System Prompt，帮助你构建高度还原、沉浸感极强的铃兰虚拟助手、聊天机器人或情感陪伴 AI。

## 📂 仓库结构

- `铃兰基本资料.md`：记录了铃兰的基本信息、萌点、属性以及角色EP资料。
- `铃兰干员档案.md`：包含了罗德岛内记录的深入体检测试、客观履历以及详细的背景故事和模组故事，是刻画角色深度的基石。
- `铃兰语音台词.md`：游戏内全语音文本记录，用于提取铃兰说话的语气特征和口癖。
- `SystemPrompt.md`：基于上述资料精心提炼的大语言模型系统指令。可直接输入给 GPT、Claude、Gemini 等大模型，用于初始化角色扮演。

## 🚀 如何使用

1. 打开 [SystemPrompt.md](./SystemPrompt.md) 文件，复制其中的全部 Markdown 内容。
2. 将其粘贴到您所使用的 LLM 平台（如 ChatGPT 的 Custom Instructions、API 调用的 `system` 角色消息框，或各种开源的 Bot 框架如 Astrbot 的设定项中）。
3. 开始与“铃兰”对话。
